# LogicDeck  
タスクの優先順位をロジックで可視化するWebアプリ

---

## 💡 解決したい課題

タスクの優先順位が感覚に依存し、  
重要なタスクが後回しになる問題があります。

---

## 🚀 解決方法

独自のスコアリングロジックにより、  
優先順位を自動で算出・可視化します。

---

## 📸 画面イメージ
![LogicDeck画面](./logicdeck.png)

---

## ✨ このアプリでできること

- タスクの登録・一覧表示  
- 独自ロジックによる優先順位の自動算出  
- 戦略レイヤーによる分類  
- 解析モードによる順位変化  
- LocalStorageによるデータ保存  

---

## 💡 このアプリの強み

主観（やりたい）と客観（やるべき）を掛け合わせ、  
「納得感のある優先順位」を提示できる点にあります。
